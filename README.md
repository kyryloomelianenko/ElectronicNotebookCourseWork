# ElectronicNotebookCourseWork
Курсова робота з дисципліни «Об'єктно-орієнтоване програмування».

Можливості програми
додавання контактів;
редагування контактів;
видалення контактів;
пошук контактів;
сортування контактів;
збереження даних у JSON;
привітання з днем народження.
Технології
C#
.NET 10.0
Windows Forms
System.Text.Json
Запуск програми
Відкрити файл курсова.sln у Visual Studio.
Натиснути кнопку Start.
